# Checklist de Requisitos

- Área ERP seleccionada: **Ventas**.
- Ventas actúa como único observador del módulo Inventario.
- Se aplica el patrón de diseño **Observer**.
- La solución tiene interfaz gráfica basada en formularios Java Swing.
- Permite registrar productos con código, descripción, categoría, stock y precio.
- Permite actualizar stock.
- Permite modificar precio.
- Permite registrar nuevos productos.
- Permite desactivar productos existentes.
- Genera notificaciones automáticas por:
  - Nuevos productos.
  - Actualización de stock.
  - Cambio de precio.
  - Desactivación de producto.
- Muestra historial detallado de notificaciones.
- Muestra resumen de operaciones realizadas y notificaciones enviadas.
- Incluye personalización visual con logo textual y colores institucionales de **TechNova Software**.
- Incluye arquitectura de software.
- Incluye diagrama de secuencia a nivel de diseño.
- Incluye diagrama de clases.
